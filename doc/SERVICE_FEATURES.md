# PC Service Features

This document summarizes the PC-side service features added to the simulator.

## Included Services

- `HTTP`
- `DNS`
- `DHCP`
- `FTP`
- `Mail`
- `NTP`

## FTP

- FTP service settings are available in the PC Services tab.
- Password visibility can be toggled with an eye icon.
- The file list is shown in the FTP tab.
- Files can be deleted from the list.

## Mail

- Mail service settings are available in the PC Services tab.
- Password visibility can be toggled with an eye icon.
- Mail server fields include SMTP, POP3, domain, username, and password.

## NTP

- NTP settings are available in the PC Services tab.
- The tab stores server and timezone values.

## CLI

- `ftp` and `mail` are available in the CLI command set.
- Both commands are included in autocomplete suggestions.
