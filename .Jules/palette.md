## 2025-05-14 - [Keyboard Shortcut Hints in Terminal]
**Learning:** Adding keyboard shortcut hints (Ctrl+F, Ctrl+L) to tooltips using a dedicated `ShortcutBadge` component significantly improves the discoverability of advanced features for power users while keeping the UI clean.
**Action:** Use `ShortcutBadge` with `scale-75` inside `TooltipWrapper` content for icon-only buttons that have keyboard equivalents.
